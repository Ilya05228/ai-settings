## response-language

Локальный плагин для автоматического выбора языка ответа, если пользователь не указал его явно.

Что входит:
- skill `response-language-selector`
- стартовые evals для проверки логики выбора языка

Основные правила:
- язык ответа выбирается по доминирующему языку самой задачи
- порог выбора: больше `80%`
- служебные куски, slash-команды, упоминания skills, поиск, ссылки, код, пути и другой шум не должны влиять на выбор
- код всегда писать на английском
- комментарии в коде писать на выбранном языке ответа

Установка в Cursor:

```bash
./commands install --dest ./.cursor --plugins response-language
```
