# prompt-engineer

Синкается из [Jeffallan/claude-skills](https://github.com/Jeffallan/claude-skills) (`skills/prompt-engineer/`), лицензия MIT.

Обновление локальной копии:

```bash
./commands sync-plugin prompt-engineer
```

Установка в Cursor:

```bash
./commands install --dest ./.cursor --plugins prompt-engineer
```
