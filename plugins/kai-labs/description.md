## kai-labs

Локальный плагин для помощи с лабораторными работами КАИ.

Что входит:
- skill `kai-lab-assistant`
- шаблон отчета `assets/kai-lab-report-template.docx`
- стартовые evals для проверки поведения skill

Основные правила:
- сначала уточнять задачу, контекст и входные материалы
- отдельно выяснять, что именно нужно: код, отчет, Excel, пояснение, презентация, PDF или комбинация
- если язык программирования не указан, по умолчанию брать `Python`
- отчет готовить в `DOCX`
- в отчете всегда добиваться наличия разделов `Цель`, `Задачи` и `Вывод`
- использовать профильные skills для `docx`, `xlsx`, `pptx` и `pdf`, когда это уместно

Установка в Cursor:

```bash
./commands install --dest ./.cursor --plugins kai-labs
```
